# Phupha: Thai Word Frequency Dataset
Phupha is A Thai Word Frequency Dataset from Common Crawl Corpus

We use [Infini-gram mini API](https://infini-gram-mini.readthedocs.io/en/latest/api.html) to query word count from CommonCrawl Corpus (Common Crawl July 2025 Crawl).

File:
- wordfreq.csv: Thai Word Frequency Dataset
- make.py: Create Thai Word Frequency Dataset
- requirements.txt: Python requirements.txt
